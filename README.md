# Smallest and Second Smallest Element Finder (C Program)

## 📌 Description
This program finds the smallest and second smallest elements in an array.

## 🚀 How It Works
- Assume the first element is the smallest.
- Traverse the array.
- Update the smallest and second smallest values when needed.
- Display the results.

## 📥 Input Array

```text
50 20 80 10 5 70
```

## 📤 Output

```text
Smallest = 5
Second Smallest = 10
```

## 🛠️ Concepts Used
- Arrays
- Loops
- Conditional Statements
- Minimum Value Search

## 🎯 Learning Outcome
- Finding minimum values in arrays
- Tracking two smallest elements
- Improving array problem-solving skills
